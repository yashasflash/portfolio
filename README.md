⚡ About This Project

This repository contains the source code for my personal portfolio website. Designed with a modern, cyberpunk-inspired aesthetic, it serves as a central hub to showcase my skills in Web Development, IoT & Automation, and Python Backends.

The entire project is built into a highly optimized, single-page application using pure HTML, Tailwind CSS, and Vanilla JavaScript, ensuring lightning-fast load times and seamless animations.

✨ Key Features

Interactive 3D Background: Immersive Spline 3D scene integrated natively with Aceternity-style SVG spotlight effects.

Dynamic Theme Switcher: 4 custom themes (Cyberpunk Cyan, Neon Purple, Matrix Green, Crimson Red) that update CSS variables and UI elements in real-time. LocalStorage enabled to remember user preferences.

Cinematic Scroll Reveals: Elements gracefully slide and fade into view as you scroll down the page, utilizing the Intersection Observer API.

Orbiting Tags Modal: A custom-built, perfect-circle pop-up modal featuring a unique 3D orbiting animation for tech stack tags.

Fully Responsive: Flawless layout across all devices, from ultra-wide monitors to mobile screens, featuring a custom glass-morphism mobile navigation menu.

Typewriter Effect: A custom Vanilla JS typewriter effect highlighting my core disciplines.

🛠️ Tech Stack

Structure: HTML5

Styling: Tailwind CSS (via CDN), Custom CSS Variables, Glassmorphism (Backdrop-filter)

Interactivity: Vanilla JavaScript (ES6+)

3D Graphics: Spline Viewer Web Component

Icons: Boxicons

🚀 Quick Start (Local Setup)

Since this project is built with standard web technologies without heavy build steps, running it locally is incredibly simple.

Clone the repository:

git clone https://github.com/yashasflash/portfolio.git


Navigate to the directory:

cd portfolio


Run the project:
Simply double-click the index.html file to open it in your default web browser, or use an extension like VS Code's Live Server for hot-reloading.
